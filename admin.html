How to add a template

  1. Fill in the details and pick a thumbnail image below
  2. Click Generate JSON Entry
  3. Copy the output and paste it into library.json (inside the [ ] array)
  4. Upload your PSD to /psds/your-id.psd in the GitHub repo
  5. Upload the thumbnail to /thumbs/your-id.jpg in the GitHub repo
  6. Commit — done
