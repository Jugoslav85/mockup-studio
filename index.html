What is an input?Each column = one set of mockups. One image per slot, one output per template. Add more inputs to generate variations side by side.
⚡ Auto-fillDistributes your uploaded images across all templates automatically. For single-slot templates, one image per column. For multi-slot templates, images are grouped — e.g. images 1–3 fill Input 1, images 4–6 fill Input 2.
＋ Add inputCreates a new empty column so you can assign a different set of images to the same templates — useful for comparing variations or processing multiple clients.
Drag & dropDrag any image thumbnail directly into a slot. Or click an empty slot to pick a file from your computer.

Why stay on this tab?

Scaylr processes your designs using a rendering engine that runs entirely inside your browser tab. This engine needs to actively open files, inject your designs, and export each mockup — one at a time.

When you switch to a different tab, browsers automatically slow down or suspend background tabs to save resources. This causes the rendering engine to stall, which is why we automatically pause the batch when you leave.

Everything resumes the moment you come back — no work is lost.

💡 Tip: You can leave the tab open in a separate window while you work in another — as long as the tab itself stays visible, processing continues at full speed.
